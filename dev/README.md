## Instruções para rodar o app localmente:

1. Inicie o MongoDB

```bash
mongod
```

2. Instale as dependências:
```bash
npm install
```

3. Execute o app:
```bash
npm start
```
## Instruções para executar os testes:

```bash
npm test
```
